# 🎣 星塘物语：猫咪的钓鱼之旅

一个有趣的网页钓鱼游戏，玩家可以体验在小猫陪伴下钓鱼的乐趣！

## 🎮 游戏玩法

### 猫咪养成
1. 前往小猫页面，查看猫咪状态
2. 钓到的食物可以喂给猫咪
3. 抚摸猫咪提升好感度
4. 照顾好猫咪让它健康成长

### 钓鱼
1. 点击 **🎣 抛竿** 按钮开始钓鱼
2. 等待鱼儿上钩（鱼漂会抖动并变红）
3. 出现提示时点击 **⚡ 提竿**
4. 运气好可以钓到稀有物品！

## ✨ 游戏特色

### 🐱 猫咪养成系统
- **喂食** - 用钓到的食物喂猫
- **抚摸** - 点击猫咪提升好感度
- **命名** - 可以给猫咪起名字
- **状态显示** - 饥饿值和好感度实时显示

### 📖 故事背景
- 精美的开场动画，讲述小猫"刀疤"的冒险故事
- 在小猫页面点击"📖 故事"查看完整剧情

### 🌤️ 天气系统
- ☀️ **晴天** - 蝴蝶飞舞
- 🌧️ **雨天** - 雨滴特效，青蛙、蜗牛等出现概率提升
- 🌙 **夜晚** - 星星、萤火虫、月亮倒影
- ☁️ **阴天** - 云朵飘动

### ⭐ 稀有度系统
| 稀有度 | 颜色 | 说明 |
|--------|------|------|
| 🟢 普通 (N) | 灰色 | 常见物品 |
| 🔵 稀有 (R) | 蓝色 | 较少出现 |
| 🟣 史诗 (SR) | 紫色 | 难得一见 |
| 🟡 传说 (SSR) | 金色 | 极其稀有 |

### 🎯 每日挑战
- 每天随机抽取一个挑战任务
- 完成挑战获得 **金色鱼漂**
- 金色鱼漂可必钓到史诗/传说物品！

## 🚀 如何运行

### 本地运行
```bash
# 方式1：双击 start.bat
start.bat

# 方式2：手动运行
python server.py
# 然后访问 http://localhost:8081
```

### 在线部署

#### 方法1：GitHub Pages
1. 将代码推送到 GitHub 仓库
2. 进入 Settings → Pages
3. Source 选择 "main" branch
4. 保存后会生成网址

#### 方法2：Netlify
1. 登录 Netlify
2. 导入 GitHub 仓库
3. 自动部署，生成网址
