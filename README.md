# 🎣 钓鱼游戏 (Fishing Game)

一个有趣的网页钓鱼游戏，玩家可以体验抛竿、等待鱼儿上钩、提竿收获的乐趣！

## 🎮 游戏玩法

1. 点击 **🎣 抛竿** 按钮开始钓鱼
2. 等待鱼儿上钩（鱼漂会抖动并变红）
3. 出现提示时点击 **⚡ 提竿**
4. 运气好可以钓到稀有物品！

## ✨ 游戏特色

### 天气系统
- ☀️ **晴天** - 蝴蝶飞舞
- 🌧️ **雨天** - 雨滴特效，青蛙、蜗牛等出现概率提升
- 🌙 **夜晚** - 星星、萤火虫、月亮倒影
- ☁️ **阴天** - 云朵飘动

### 稀有度系统
| 稀有度 | 颜色 | 说明 |
|--------|------|------|
| 🟢 普通 | 灰色 | 常见物品 |
| 🔵 稀有 | 蓝色 | 较少出现 |
| 🟣 史诗 | 紫色 | 难得一见 |
| 🟡 传说 | 金色 | 极其稀有 |

### 每日挑战
- 每天随机抽取一个挑战任务
- 完成挑战获得 **金色鱼漂**
- 金色鱼漂可必钓到史诗/传说物品！

### 彩蛋
- 第三次钓鱼有机会钓到神秘作者 👧

## 🚀 如何部署

### 方法1：GitHub Pages
1. 将代码推送到 GitHub 仓库
2. 进入 Settings → Pages
3. Source 选择 "main" branch
4. 保存后会生成网址

### 方法2：Netlify
1. 登录 Netlify
2. 导入 GitHub 仓库
3. 自动部署，生成网址

## 📝 技术栈

- CSS3 (动画、渐变、阴影)
- Vanilla JavaScript
- LocalStorage (数据保存)
